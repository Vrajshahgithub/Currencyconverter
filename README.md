💱 Currency Converter

A simple and responsive Currency Converter built with React.js that fetches real-time exchange rates using a Currency API and Fetch API. It demonstrates API integration, props, state management, and custom hooks to perform fast and accurate currency conversion.

🛠️ Tech Stack
React.js
JavaScript (ES6+)
Tailwind Css
Fetch API
Currency Exchange API

📌 How It Works
The application fetches the latest currency exchange rates from a public Currency API using the Fetch API.
Users select the source and target currencies, enter an amount, and the application instantly calculates the converted value.
