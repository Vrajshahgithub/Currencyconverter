<h1 align="center">💱 Currency Converter</h1>

<p align="center">
A simple and responsive Currency Converter built with <b>React.js</b> that fetches real-time exchange rates using a Currency Exchange API.
It demonstrates <b>API integration, Props, State Management, Custom Hooks,</b> and <b>Fetch API</b> for fast and accurate currency conversion.
</p>

---

<h2>🚀 Tech Stack</h2>

-  React.js
-  JavaScript (ES6+)
-  Tailwind CSS
-  Fetch API
- Currency Exchange API

---

<h2>✨ Features</h2>

- Real-time currency conversion
- Supports multiple currencies
- Instant conversion on input
- Responsive UI
- Reusable React components
- Custom Hook for fetching exchange rates

---

<h2>📌 How It Works</h2>

1. Fetches the latest exchange rates using a Currency Exchange API.
2. Users select the **From** and **To** currencies.
3. Enter the amount to convert.
4. The application instantly calculates and displays the converted value.
